# ERRATA - Project1D

## Current Release

### Achievements

- Implemented a complete block-based module builder with 12 financial education block types.
- Added drag-and-drop ordering, live preview, JSON import/export, and 50-step undo/redo.
- Added deterministic financial calculators and simulated review panels.
- Added unit tests for calculator formulas, review scoring, and store history behavior.

### Known Limitations

- The media block uses an internal placeholder instead of a real media asset browser.
- Quiz editing is currently handled through metadata JSON rather than a dedicated option editor.
- Collaborative editing and backend persistence are intentionally out of scope for the local submission build.

### Planned Enhancements

1. Add a visual quiz option editor.
2. Add autosave and backend draft persistence.
3. Add PDF/SCORM export.
4. Add granular role permissions for 50+ creators and reviewers.
5. Add Playwright visual regression tests for desktop and mobile previews.
