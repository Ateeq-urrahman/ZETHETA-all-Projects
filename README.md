# EduForge Interactive Financial Education Module Builder

Production-grade visual editor for a fictional Series A fintech startup, EduForge Financial Technologies. The app lets non-technical content creators assemble portable financial education modules without engineering support.

## Highlights

- 12 block types: hero, rich text, callout, quiz, EMI calculator, SIP calculator, compound interest, animated explainer, progress tracker, achievement badge, video embed, glossary.
- Drag-and-drop block reordering through dnd-kit.
- TipTap rich text editing for lesson copy.
- Zustand state model with 50-step undo and redo.
- Live preview with desktop and mobile toggles.
- JSON export/import for backend portability.
- Framer Motion animated concept explainers and responsive preview transitions.
- Simulated Content Creator Panel and Technical Architecture Review Board readiness scoring.
- WCAG 2.1 AA-minded semantics, labels, focus states, and keyboard-accessible controls.

## Run Locally

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Verification
