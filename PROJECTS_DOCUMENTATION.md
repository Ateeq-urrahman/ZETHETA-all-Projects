# ZETHETA Project Portfolio Documentation

This repository contains a collection of modern web applications built with Next.js, React, TypeScript, and related frontend tooling. The projects were developed as separate portfolio pieces and are organized under a single repository for easy review and presentation.

## Repository Overview

The repository includes five distinct projects:

1. Root Project - EduForge Interactive Financial Education Module Builder
2. Project 1A - Zetha Stock Screener
3. Project 1B - LendSwift Loan Application Wizard
4. Project 1C - Meridian Capital Portfolio Analytics Dashboard
5. Project 1D - Placeholder project folder for future expansion

## 1. Root Project: EduForge Interactive Financial Education Module Builder

### Purpose
A visual editor for building reusable financial education modules. It allows non-technical content creators to assemble lesson content using drag-and-drop blocks, rich text editing, and preview tools.

### Key Features
- Drag-and-drop module construction
- Rich text content editing
- Multiple content block types including calculators and quizzes
- Live preview with desktop and mobile modes
- JSON import/export support
- Undo/redo support and state persistence

### Tech Stack
- Next.js
- React
- TypeScript
- Zustand
- TipTap
- dnd-kit
- Framer Motion

### Run Locally
```bash
npm install
npm run dev
```

### Notes
This project is the most complete of the portfolio pieces and is designed to showcase architecture, interactivity, and polished UI.
